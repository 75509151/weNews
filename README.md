# weNews

pyramid + react native


# App

##### React Native:#####
```
$ npm i -g react-native-cli
```

##### clone & install this repo: #####

```
$ git clone https://github.com/75509151/weNews.git
$ cd weNews/app/weNews
$ sudo npm install
```

##### Android :#####
```
$ npm start (cd weNews/app/weNews)

$ react-native run-android

```
